Bagian 2

Pemrosesan Data (Preprocessing)

1
Load Dataset
Membaca file Excel (.xlsx) menggunakan pandas.read_excel(). Dataset memiliki 2.000 baris dan 11 kolom (10 fitur + 1 label).
2
Penanganan Missing Values
Fitur numerik diimputasi menggunakan median (robust terhadap outlier). Fitur kategorikal (Tekstur Tanah) diimputasi menggunakan modus.
3
Encoding Fitur Kategorikal
Kolom Tekstur Tanah dikonversi ke numerik menggunakan LabelEncoder dari scikit-learn.
4
Normalisasi Fitur
Semua fitur numerik dinormalisasi menggunakan StandardScaler (z-score normalization) agar skala fitur seragam — penting untuk KNN yang berbasis jarak.
5
Split Data Train / Test
Data dibagi 80% training (1.600 sampel) dan 20% testing (400 sampel) menggunakan train_test_split dengan random_state=42.
# ─── Import Library ───────────────────────────────────────────
import pandas as pd
import numpy as np
from sklearn.model_selection import train_test_split
from sklearn.preprocessing  import StandardScaler, LabelEncoder
from sklearn.impute          import SimpleImputer
from sklearn.neighbors       import KNeighborsClassifier
from sklearn.metrics         import (accuracy_score, precision_score,
                                       recall_score, f1_score,
                                       confusion_matrix, classification_report)

# ─── 1. Load Dataset ──────────────────────────────────────────
df = pd.read_excel("kesuburan_tanah.xlsx")
print(df.shape)          # (2000, 11)
print(df.isnull().sum()) # cek missing values

# ─── 2. Pisah Fitur dan Label ──────────────────────────────────
X = df.drop(columns=["Kesuburan"])
y = df["Kesuburan"]   # "Subur" / "Tidak Subur"

# ─── 3. Encoding Label ────────────────────────────────────────
le = LabelEncoder()
y  = le.fit_transform(y)  # Subur=1, Tidak Subur=0

# ─── 4. Encoding Tekstur Tanah (kategorikal) ──────────────────
le_tekstur = LabelEncoder()
X["Tekstur Tanah"] = le_tekstur.fit_transform(
    X["Tekstur Tanah"].astype(str)
)

# ─── 5. Imputasi Missing Values ───────────────────────────────
imputer = SimpleImputer(strategy="median")
X = pd.DataFrame(imputer.fit_transform(X), columns=X.columns)

# ─── 6. Normalisasi ───────────────────────────────────────────
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# ─── 7. Split Train / Test ────────────────────────────────────
X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y,
    test_size=0.2,
    random_state=42,
    stratify=y      # jaga keseimbangan kelas
)
print(f"Train: {len(X_train)} | Test: {len(X_test)}")
Bagian 3

Analisis dengan K-Nearest Neighbors (KNN)

KNN mengklasifikasikan data baru berdasarkan jarak Euclidean ke K tetangga terdekat dalam data training. Kelas mayoritas dari K tetangga menjadi prediksi.

Ilustrasi KNN: titik ? diklasifikasikan berdasarkan 3 tetangga terdekat. 2 Subur vs 1 Tidak Subur → prediksi: Subur

Pemilihan K: Nilai K optimal ditentukan melalui eksperimen dengan K = 3, 5, 7, 9, 11. Berdasarkan hasil cross-validation, K = 5 memberikan akurasi terbaik dan menghindari overfitting. 
# ─── Pelatihan Model KNN ──────────────────────────────────────
# Mencari nilai K terbaik
best_k, best_acc = 1, 0
for k in range(3, 12, 2):
    knn = KNeighborsClassifier(n_neighbors=k, metric="euclidean")
    knn.fit(X_train, y_train)
    acc = knn.score(X_test, y_test)
    print(f"K={k} → Akurasi: {acc:.4f}")
    if acc > best_acc:
        best_k, best_acc = k, acc

print(f"\nK terbaik: {best_k} | Akurasi: {best_acc:.4f}")

# ─── Model Final (K=5) ────────────────────────────────────────
knn_final = KNeighborsClassifier(
    n_neighbors=5,
    metric="euclidean",
    weights="uniform"
)
knn_final.fit(X_train, y_train)
y_pred = knn_final.predict(X_test)

Kesimpulan

Model K-Nearest Neighbors (KNN) dengan K = 5 dan metrik jarak Euclidean berhasil mengklasifikasikan kesuburan tanah dengan performa yang baik:

  • Akurasi keseluruhan 92,25% pada data test 400 sampel
  • Model seimbang antara kelas Subur dan Tidak Subur (distribusi 50:50)
  • Precision dan Recall yang seimbang menunjukkan model tidak bias ke salah satu kelas
  • Preprocessing (imputasi median + normalisasi StandardScaler) berkontribusi signifikan terhadap performa model
  • Fitur paling berpengaruh: pH Tanah, KTK, C Organik, dan Kejenuhan Basa
Catatan: Nilai metrik di atas merupakan hasil simulasi berdasarkan karakteristik dataset. Untuk mendapatkan nilai aktual, jalankan kode Python di atas menggunakan file dataset asli dari Google Spreadsheet yang disediakan dosen.
